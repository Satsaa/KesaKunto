# `showcase` Internal

Single Vite-based showcase app for KesaKunto.

## Scope

- This is the only app under `apps/`.
- Keep it focused on quickly previewing package-level UI and app concepts.
- Prefer simple React code and local state.

## Tech

- Vite
- React
- React Compiler via the Vite React plugin Babel config

## Rules

- Do not add Expo or React Native scaffolding here.
- Keep the app small and easy to reset.
- If the showcased surface changes materially, update `FEATURES.md` when product behavior changes.
